# Trading-Sandbox

This is an interactive trading sandbox

What this project includes  
- Stochastic price simulation for market movement  
- Long and short position trading  
- Real-time profit and loss tracking  
- Streamlit interface for interacting with the system  

## Live App
https://trading-sandbox-brian-young.streamlit.app/

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
